macOS Installation Guide

Since the EU Financial Planner is a self-built open-source app, follow these steps to open it securely on macOS.

01

Download the Installer

▼

Click the main download button at the top of this page or download the latest `.dmg` installer directly from the Releases Page.

02

Install to Applications

▼

Double-click the downloaded `.dmg` file to mount it, then drag the **EU Financial Planner.app** icon directly into your **Applications** folder.

03

Bypass Gatekeeper (First Launch Only)

▼

Because the app is compiled directly from source and is unsigned, macOS will block launch with an "Unidentified Developer" prompt on double-click.

To open it:

  1. Go to your **Applications** folder in Finder.
  2. Right-click (or Control-click) **EU Financial Planner.app** and select **Open**.
  3. In the dialog that appears, click the **Open** button.
  4. This registers an exception and the app will open normally via double-click next time!

Developer Terminal Workaround:

Alternatively, you can open your terminal and run the following command to remove the macOS quarantine flag directly:

xattr -d com.apple.quarantine /Applications/EU\ Financial\ Planner.app
